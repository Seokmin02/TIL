# List의 다양한 기능

### List의 기능
- list.index(value)     # 값을 이용하여 위치를 찾는 기능
- list.extend([value1, value2, ...])        # 리스트 뒤에 값을 추가
- list.insert(index, value)       # 원하는 위치에 값을 추가
- list.sort()       # 값을 순서대로 정렬
- list.reverse()        #값을 역순으로 정렬

### List와 문자열
- list = str.split()        # 문자열에서 리스트로
- " ".join(list)         # 리스트에서 문자열로
